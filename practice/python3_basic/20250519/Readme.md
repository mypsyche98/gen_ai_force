  * import 모듈명 as 모듈명별명
  * from 모듈명 import 함수명 or *

----

## 예외처리
* 특정 코드 블럭내에 발생할지도 모르는 또는 실행시 오류가 발생하는 예외에 대한 개별 처리코드
```
try:
  실행코드
except 에러이름 as 메세지변수명:
  에러발생할시 실행할코드
else: # else구문은 생략가능
  에러가 아닌경우 실행할코드
finally:
  except/else 처리후 무조건 실행될코드
```
